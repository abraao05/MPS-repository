# Registro de Verificação de GQA

> **O que é este documento.** Registro de uma verificação de Garantia da Qualidade realizada sobre o processo e os produtos de trabalho de um projeto. Um registro por verificação. O auditor de GQA deve ser independente da equipe verificada.
>
> **Como usar.** Preencha os campos marcados `[preencher]`. Nas tabelas de verificação, use ✅ (Conforme), ⚠ (Desvio) ou N/A (Não aplicável). Para cada ⚠, registre um achado na seção 4. Se nenhum desvio foi encontrado, registre explicitamente "Nenhum desvio identificado" — nunca deixe a seção vazia.
