# Plano de Gerência de Configuração

> **O que é este documento.** Define como a integridade dos produtos de trabalho é mantida e como eles são disponibilizados a todos os envolvidos. Cobre: o que é controlado (itens de configuração), como mudanças são controladas, como baselines são marcadas, como o histórico é mantido e como auditorias verificam a integridade.
>
> **Como usar.** É majoritariamente definição estável (seções 1–2) + registro vivo de baselines e auditorias (seções 3–5). `‹padrão›` vem da organização.
