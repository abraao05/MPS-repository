## 3. Discussões e definições

Revisado o escopo completo do Plano v1. A equipe apresentou as estimativas de dimensão (120 pontos) e duração (~10 semanas) com base na velocidade de referência observada no próprio projeto.

Sobre o módulo de comércio (pagamentos): o cliente confirmou que a integração com o gateway de pagamento tem pré-requisitos não atendidos no momento (conta de empresa, integração com emissor). Decidiu-se parquear o módulo para a v2.0 para não impactar o cronograma do núcleo. Esta decisão foi formalizada via Change Request CR-TCON-001.

O orçamento de R$ 45.000 *(exemplo)* foi confirmado pelo cliente, com alocação por fase conforme seção 4.2 do Plano.
