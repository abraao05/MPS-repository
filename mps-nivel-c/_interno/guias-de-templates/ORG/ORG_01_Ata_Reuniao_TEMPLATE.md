# Ata de Reunião — [ASSUNTO]

> **O que é este documento.** Registro formal de uma reunião relevante: o que foi discutido, o que foi decidido, as ações acordadas e os próximos passos. Aplicável a qualquer tipo de reunião com impacto no projeto — kickoff, aprovação do plano, sprint review, reunião com cliente, análise crítica, encerramento.
>
> **Como usar.** Preencha os campos marcados `[preencher]`. A seção mais importante é a 4 (Decisões e aceites) — é o que transforma a conversa em compromisso rastreável. Uma ata sem decisões registradas não serve como evidência de marco.
