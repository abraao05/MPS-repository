# Plano de Medição

> **O que é este documento.** Define os objetivos de medição da organização (o que se quer saber/melhorar) e como eles se ligam às medidas do Repositório. Enquanto o Repositório guarda as medidas e os dados, este Plano guarda o *porquê* — os objetivos de negócio que a medição serve.
>
> **Como usar.** A seção 1 (objetivos) deriva dos objetivos de negócio da organização — só a liderança da Timeware pode defini-los de verdade. Os exemplos são marcados como tal, para substituir.
