# Relatório de Acompanhamento do Projeto

> **O que é este documento.** Registro periódico do andamento do projeto: o que aconteceu de fato, comparado ao que o *Plano do Projeto* combinou. Enquanto o Plano muda por evento (replanejamento, mudança de escopo), este relatório muda por **cadência** — é atualizado a cada ciclo de acompanhamento (ex.: a cada fase concluída ou a cada período definido). Ele mostra desvios, ações tomadas e o que se aprendeu.
>
> **Como usar.** Este documento sempre se refere a uma versão específica do Plano (o baseline). Preencha a cada ciclo. Não repita o conteúdo do Plano — referencie-o. O foco aqui é **planejado vs. realizado** e o tratamento dos desvios.
