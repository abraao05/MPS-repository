# Termo de Abertura do Projeto — [preencher: NOME DO PROJETO]

> **O que é este documento.** Registro formal que autoriza o início do projeto. Ele precede qualquer estimativa detalhada ou planejamento — seu papel é dizer "este projeto existe, tem um objetivo claro, e há um responsável nomeado". É a base sobre a qual o Plano do Projeto será elaborado.
>
> **Como usar.** Preencha os campos marcados `[preencher]` assim que a decisão de iniciar o projeto for tomada. Seções que dependem de informações ainda a confirmar usam `[a confirmar]` — mas o TAP deve ser emitido antes do kickoff, não depois. Um TAP incompleto é melhor que nenhum TAP.
>
> **Regra de ouro.** O Termo de Abertura não é o Plano do Projeto. O objetivo aqui é macro: quem, o quê e por quê — não como, quanto ou com que detalhe. O detalhamento vem no Plano, que parte daqui.
