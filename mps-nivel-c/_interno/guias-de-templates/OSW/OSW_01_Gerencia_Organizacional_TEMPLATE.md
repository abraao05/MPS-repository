# Gerência Organizacional de Software

> **O que é este documento.** Define como a gerência da organização governa o conjunto — alinhando objetivos de negócio, processos, recursos e projetos. É o nível mais alto: enquanto os processos de projeto cuidam de um projeto, este cuida da organização como um todo e do portfólio de projetos.
>
> **Modelo da organização (importante).** A Timeware é fábrica de software / alocação de squads, sem corpo fixo: squads são contratadas e alocadas conforme o volume vendido. Consequências para este processo: (a) a priorização do portfólio é dada por **venda + viabilidade de execução** (a viabilidade pode barrar a venda); (b) **não há disputa de recursos entre projetos** (squads vêm sob demanda), então a governança de portfólio foca em alinhamento e acompanhamento agregado, não em arbitrar conflitos; (c) a liderança já acompanha o desempenho de forma agregada.
